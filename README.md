# Cards challenge

## Prototype

- [Figma](https://www.figma.com/file/gKik6TRqyZOfIdyibFSn0c/Cards-Interview?node-id=2%3A2)

## Stack

- Typescript
- Expo

## Installation

- Install expo

```
yarn global add expo-cli
```

- Clone the repo

```
git clone git@github.com:SofiaChuquin/cards-challenge.git
```

- Access the repo

```
cd cards-challenge
```

- Start the project in expo

```
yarn install
yarn start
```
